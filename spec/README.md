## Quick Reference

- **App**: Interactive piano playable via computer keyboard
- **Range**: 4 octaves (C1–B4), displayed 2 at a time
- **Audio**: Web Audio API with @tonaljs for frequencies
- **Stack**: Next.js 16, React 19, TypeScript, Tailwind CSS
