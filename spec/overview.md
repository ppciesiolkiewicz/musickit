# Music Kit — Overview

## Product Description

Music Kit is a web-based interactive piano application. Users can play a virtual piano using their computer keyboard, with real-time audio feedback and music theory visualization.

## Goals

- **Accessibility**: Play piano without physical instrument or MIDI hardware
- **Education**: Visualize scales and learn note names
- **Simplicity**: Intuitive keyboard mapping, no installation required

## Target Users

- Music students learning piano
- Developers experimenting with audio/theory
- Anyone wanting to play piano in the browser

## Core Value Proposition

A minimal, responsive piano that runs in the browser with accurate sound, scale highlighting, and a familiar keyboard layout that maps the home row to piano keys.
